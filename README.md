# RalphX Homebrew Tap

Custom Homebrew tap for RalphX desktop.

Status:
- repository is bootstrapped
- the `ralphx` cask will be added after the first signed and notarized release assets are published

Planned install flow:
```bash
brew tap aigentive/homebrew-ralphx
brew install --cask ralphx
```

This repository contains Homebrew metadata only. Release binaries live in `aigentive/ralphx-releases`.
